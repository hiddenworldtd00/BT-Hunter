# BT_Hunter - Bluetooth Device Hunter

**Développé par HiddenWorld Communauté Tchadienne**

BT_Hunter est un outil avancé de surveillance, d'analyse et de gestion des appareils Bluetooth environnants. Il permet de scanner, identifier, signaler et contrôler les connexions Bluetooth à proximité.

## ⚠️ Avertissement Légal

**Usage strictement légal et éthique :**
- ✅ Surveiller vos propres appareils Bluetooth
- ✅ Tester la sécurité de vos propres périphériques
- ✅ Rechercher vos appareils perdus
- ✅ Audit de sécurité autorisé

**INTERDICTIONS :**
- ❌ Désautoriser des appareils sans permission
- ❌ Intercepter des communications privées
- ❌ Toute activité malveillante

## Fonctionnalités (15+)

### Surveillance
- Scan continu des appareils Bluetooth
- Détection des changements de présence
- Alertes sonores et visuelles
- Historique des détections

### Contrôle
- Déconnexion automatique des appareils suspects
- Bloquer/débloquer des appareils
- Forcer la déconnexion
- Protection de vos propres appareils

### Analyse
- Informations détaillées (RSSI, classe, services)
- Cartographie de proximité
- Historique de connexion
- Export des données

## Installation

```bash
pip install -r requirements_BT_Hunter.txt
sudo python3 BT_Hunter.py
```

## Utilisation

1. Lancez avec sudo pour les fonctionnalités avancées
2. Choisissez une option dans le menu
3. Pour la surveillance continue, utilisez l'option 1
4. Pour la déconnexion auto, configurez les filtres (option 5)

## Prérequis

- Linux avec Bluetooth
- Python 3.8+
- Droits root pour certaines fonctionnalités
- bluez, hcitool, bluetoothctl
